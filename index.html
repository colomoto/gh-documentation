Some documentation

MDDlib

MDDlib is a simple java toolkit for MDD manipulation.

For now, only API documentation (javadoc) is available.

Logical Models

LogicalModel is a simple java toolkit for the definition and manipulation of logical models. It relies on MDDlib for the logical functions. It is designed to share core data structure between external tools, which have to provide a nice GUI to edit the models and run the included actions.

For now, only API documentation (javadoc) is available.
